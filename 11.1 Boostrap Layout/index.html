Como Inserir e Usar Bootstrap

Passo 1: Incluindo o Bootstrap

Para começar a usar o Bootstrap, inclua o link para o arquivo CSS do Bootstrap em sua página HTML, dentro da tag <head>:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">

Passo 2: Criando um Container

Use a classe .container para criar um contêiner responsivo para envolver o conteúdo da página e centralizá-lo horizontalmente:

<div class="container">
    
</div>

Passo 3: Adicionando Componentes

Use os componentes do Bootstrap, como cards, para estruturar o conteúdo da sua página de forma visualmente atraente:

Passo 4: Personalizando Estilos

Além das classes Bootstrap padrão, você pode adicionar estilos personalizados usando classes CSS personalizadas:

.custom-bg {
    background-color: lightgray;
    padding: 20px;
}
